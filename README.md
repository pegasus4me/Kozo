# Kozo
chatting platform 

<h1>coté back</h1>
- nextjs server side
- MongoDB
- Prisma orm 
 <h1>coté front</h1>
    - NextJS
    - NextAuth
    - ReactIcons
    - Axios pour nos requetes http
    - SWR pour les data feching
    - Pusher pour la gestion du Live chat
    - Redux pour le state managment
