# Kozo
chatting platform 

<h1>stack used for back</h1>
- nextjs server side
- MongoDB
- Prisma orm 
 <h1>stack used for front</h1>
    - NextJS
    - NextAuth
    - ReactIcons
    - Axios for data feching
    - SWR for data cache
    - Pusher to manage Live chat
    - Redux for state managment
